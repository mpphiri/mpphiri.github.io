# Moses Patson Phiri — Website (Jekyll + Markdown)

This folder is a ready-to-upload GitHub Pages (Jekyll) site scaffold.

## Quick start (GitHub Pages)
1. Create a repo named `mosespatsonphiri.github.io` (recommended) or any repo.
2. Upload everything in this folder to the repo root.
3. GitHub: Settings → Pages → Deploy from branch → `main` / `/root`.

## Replace content
- Update `_config.yml` if you want a different title/tagline.
- Replace images in `assets/img/`.
- Replace `assets/Your-CV.pdf` with your actual CV PDF (keep the filename or update links).
